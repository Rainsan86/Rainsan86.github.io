## 🌟 项目简介

欢迎来到魔卡少女樱的魔法世界！这是一个基于 **Cardcaptor Sakura** 动漫主题的 AI 聊天助手，融合了现代 AI 技术与经典的魔法少女元素。在这里，你可以与小樱一起探索 AI 的无限可能！🌸

### 🎯 核心特性

- **🎭 双重主题模式**：普通模式与魅魔模式，满足不同用户需求
- **🪄 库洛牌系统**：支持多种 AI 模型，包括 DeepSeek、自定义模型等
- **🌙 智能对话**：多轮对话记忆，让交流更加自然流畅
- **📄 文件翻译**：支持拖拽上传文件进行批量翻译
- **🔒 安全防护**：API 密钥加密存储，保护用户隐私
- **🎨 精美界面**：魔卡少女樱主题设计，动画效果丰富

## 🚀 快速开始

### 📋 系统要求

- 现代浏览器（Chrome、Firefox、Safari、Edge）
- 稳定的网络连接
- 有效的 AI API 密钥

### ⚡ 立即体验

1. **下载项目**
   ```bash
   git clone https://github.com/Rainsan86/Rainsan86.github.io.git
   cd 魔卡少女樱v3.3.0
   ```

2. **打开应用**
   ```bash
   # 直接打开 index.html 文件
   # 或使用本地服务器
   python -m http.server 8000
   # 然后访问 http://localhost:8000
   ```

3. **配置 API**
   - 在设置中输入你的 API 密钥
   - 选择你喜欢的库洛牌（AI 模型）
   - 开始你的魔法之旅！✨

## 🎯 使用指南

### 🔑 API 配置

1. **获取 API 密钥**
   - 访问 [DeepSeek](https://platform.deepseek.com/) 注册账号
   - 创建 API 密钥
   - 复制密钥备用

2. **配置模型**
   - 选择你喜欢的库洛牌
   - 输入 API 密钥
   - 设置基础 URL（可选）

3. **开始对话**
   - 在输入框中输入你的问题
   - 点击发送或按 Enter
   - 享受与小樱的对话！

### 📄 文件翻译

1. **准备文件**
   - 准备要翻译的 .txt 文件
   - 确保文件编码为 UTF-8

2. **上传文件**
   - 拖拽文件到翻译区域
   - 或点击选择文件按钮

3. **开始翻译**
   - 点击"开始翻译"按钮
   - 等待翻译完成
   - 下载翻译结果
